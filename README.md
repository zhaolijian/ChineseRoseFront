# 阅记小程序

阅记是一款专注于纸质书阅读笔记数字化管理的微信小程序。通过智能OCR识别技术和严格的隐私保护措施，结合智能笔记整理和独创的思维导图自动生成功能，为纸质书爱好者提供从笔记录入到知识体系构建的完整解决方案。

## 🌟 核心特性

### 📚 专注纸质书
- 深耕纸质书笔记场景，不同于电子书阅读器
- 支持ISBN扫描快速添加书籍信息
- 完整的书籍管理和分类功能

### 🔒 隐私保护
- 严格的数据保护措施
- 笔记内容端到端加密存储
- 本地优先的数据存储策略

### 🧠 智能识别
- 先进的OCR文字识别技术
- 支持手写笔记识别
- 多张图片智能合并功能

### 🗺️ 知识可视化
- 业内首创笔记自动生成思维导图
- 多种导图样式和布局
- 支持导图编辑和分享

### ⚡ 极简体验
- 拍照 → 识别 → 整理 → 导图，一键完成
- 直观的用户界面设计
- 流畅的操作体验

## 🚀 技术栈

### 前端技术

- **代码规范**: ESLint + Prettier

## 🛠️ 开发指南

### 环境要求
- Node.js >= 16.0.0
- 微信开发者工具
- 小程序开发账号

### 安装依赖
```bash
npm install
```

### 开发模式
1. 使用微信开发者工具打开项目
2. 配置 AppID
3. 点击编译即可预览

### 代码规范
```bash
# 代码检查
npm run lint

# 自动修复
npm run lint:fix
```

### 构建发布
1. 在微信开发者工具中点击「上传」
2. 填写版本号和项目备注
3. 提交审核

## 🔧 配置说明

### 必需配置
在使用前，需要配置以下信息：

1. **微信小程序配置**
   - 修改 `project.config.json` 中的 `appid`
   - 配置服务器域名白名单

2. **后端服务配置**
   - 修改 `app.js` 中的 `baseUrl`
   - 配置API接口地址

3. **第三方服务配置**
   - 配置OCR服务密钥
   - 配置文件存储服务

### 可选配置
- 调整UI主题色彩
- 配置页面分享信息
- 设置统计分析
