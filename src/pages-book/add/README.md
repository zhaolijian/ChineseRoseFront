# 书籍添加/编辑页面

## 功能说明

这是一个用于添加或编辑书籍信息的页面，支持以下功能：

### 主要功能
1. **新增书籍** - 通过填写表单创建新的书籍记录
2. **编辑书籍** - 修改已有书籍的信息
3. **封面上传** - 支持从相册或相机选择图片作为书籍封面
4. **ISBN扫码** - 支持扫描ISBN码（功能待实现）
5. **日期选择** - 使用日期选择器设置出版日期

### 表单字段
- 书名（必填）
- 作者
- 出版社
- ISBN（支持扫码）
- 出版日期
- 简介（最多500字）
- 封面图片

### 页面入口
1. 从书架页面点击右上角"+"图标
2. 从书架页面点击悬浮添加按钮
3. 从书籍详情页编辑（传入书籍ID）

### 技术实现
- 使用 uView UI 组件库
- 表单验证确保数据完整性
- 响应式数据管理
- 与 bookStore 集成进行数据操作
- 支持本地图片选择（后续接入COS）
